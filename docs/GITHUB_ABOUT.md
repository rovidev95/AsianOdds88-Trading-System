# Configurar el About del repositorio en GitHub

Sigue estos pasos para que la página del repo se vea bien y Google indexe mejor.

---

## Pasos

1. Abre el repo: **https://github.com/rovidev95/AsianOdds88-Trading-System**
2. En la columna derecha, debajo del nombre del repo, verás **About**. Haz clic en el **engranaje** (⚙️) para editarlo.
3. Pega la **Descripción** y la **Web** que tienes abajo.
4. En **Topics**, añade las etiquetas una a una (o las que quieras de la lista). GitHub permite hasta 20 topics.
